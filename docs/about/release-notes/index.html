Version 1.3 (2023-12-08)

  • For more information refer to Release Notes
What's New

Version 1.2 (2023-11-01)

  • Docker Image: 1.2
  • For more information refer to Release Notes
What's New
  • Updating the Typesense image version from custom operator-config.yaml
  • The publishNotReadyAddresses=True is set to support headless services discovery in stateful sets
  • Introducing liveness and startup probes.
  • Base files updated to pull Typesense version 0.25.0

Version 1.1 (2023-07-16)

  • For more information refer to Release Notes
What's New
  • Added support for StorageClass
Bug fixes
  • Configmap creation fixed
  • Exceptions are not bypassed. If any exceptions are raised, the operator does not create resources further.
